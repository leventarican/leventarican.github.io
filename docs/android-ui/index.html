Android: UI

About

Give an overview about Android UI components. Know the naming of the components.

App bar

The app bar also known as the ActionBar. Later the Toolbar was introduced. The ActionBar as usually used in combination Activity whereas the Toolbar is more flexible. An Activity in Android context is a window like in a desktop app. And a desktop app has usually a toolbar.

Menu

There are different menu types: options menu (aka. overflow menu), contextual menu and popup menu.

Navigation

With navigation the movement (back and forth) from screen to screen is meant. Android provides two system provides UI components to navigate back on press. One option is the back button in the system navigation bar. Second option is up button in the app bar.

Layout

Layout's in Android are a huge topic. By time the available layout's evolved. You should basically know to use which layout when.

  • Whats the difference ConstraintLayout to CoordinatorLayout.
  • RecyclerView is a good choice for large list items.
  • You should also know that are <include> and <merge> layout tags
