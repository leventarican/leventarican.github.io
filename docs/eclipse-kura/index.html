About

Aim of the document is to get a basic understanding of the Kura Framework.

Introduction

Per definition eclipse kura is a OSGi (modular) framework for IoT Gateways. It has even support for field protocols (OPC-UA, S7, ...).

There is also a OPC/UA server simulator (DP: Deployment Package in Kura). The simulator will expose some sensor data.
